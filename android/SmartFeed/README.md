# SmartFeed Android

Android-клиент реализован до этапа 11 ТЗ: авторизация, лента, статьи, действия,
категории, сохранённые материалы, рекомендации, аналитика, Room offline cache и
WorkManager-синхронизация событий.

Инструкция по настройке backend, `baseUrl`, сборке и demo-аккаунтам находится в
[`README_ANDROID.md`](README_ANDROID.md).
